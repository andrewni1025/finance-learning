# 金融知识站 📈

一站式金融投资知识百科，助你从小白成为投资达人。

## 内容涵盖

- 📚 股票基础知识 & 核心术语
- 🌍 美股 / 港股 / A股 三大市场详解
- 💹 做多、做空、期权、期货、ETF、CFD 等交易类型
- 🖥️ 全球交易平台推荐
- 💱 外汇与跨境交易（人民币/美元/澳元等）
- 🏗️ 大宗商品投资（黄金/原油/白银等）
- 🧠 交易策略与技术分析
- 📰 信息获取渠道
- 🛡️ 风险管理

## 部署

纯静态网站，直接用 GitHub Pages 托管：

1. Push 代码到 GitHub 仓库
2. Settings → Pages → Source 选择 `main` 分支
3. 访问 `https://<username>.github.io/FinanceLearning/`
